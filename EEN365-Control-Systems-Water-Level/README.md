# EEN365: Control Systems - Water Level Control

## Project Overview
This project involves the design and simulation of a closed-loop control system to maintain a specific water level in a tank. The system utilizes a **PID Controller** to minimize steady-state error and manage settling time.

## Technical Specifications
* **Software:** MATLAB & Simulink
* **Controller Type:** PID (Proportional-Integral-Derivative)
* **Tuned Parameters:**
    * $K_p = 2$
    * $K_i = 0.01$
    * $K_d = 0$

## Key Objectives
1.  Model the hydraulic plant dynamics in Simulink.
2.  Analyze the open-loop response to understand system stability.
3.  Apply PID tuning to achieve a stable setpoint with minimal overshoot.

## Results
The system successfully reached the desired water level setpoint with a robust response, as documented in the provided simulation reports.
