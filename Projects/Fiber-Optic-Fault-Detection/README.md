#  Fiber Optic Fault Detection using Machine Learning

##  Project Overview
Monitoring infrastructure (like pipelines or power lines) is difficult over long distances. This project uses **Optical Fiber Sensing** combined with **Machine Learning** to automatically detect and classify physical disturbances.

##  System Architecture
1. **Sensing:** Utilizing Rayleigh backscattering or Distributed Acoustic Sensing (DAS) principles.
2. **Signal Processing:** Raw optical signals are filtered and transformed (FFT/Wavelet) to extract features.
3. **Classification:** A Machine Learning model identifies the "Signature" of the fault.
