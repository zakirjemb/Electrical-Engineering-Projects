 Pipeline Fault Detection
New York University Abu Dhabi (NYUAD) | Undergraduate Research

 Overview
Development of a real-time monitoring system for detecting oil and gas pipeline anomalies using acoustic and pressure-induced signal analysis.

 Technical Highlights
Signal Acquisition: Integrated NI-DAQ hardware with LabVIEW for high-speed, real-time data logging.

Fault Detection: * Modeled acoustic signatures to differentiate between normal flow and leakage anomalies.

Used MATLAB for post-processing and physics-based signal verification.

Systems Integration: Developed a synchronized pipeline for multi-sensor data fusion to reduce false alarms.

 Tech Stack
Hardware: DAQ Systems, Pressure Sensors, Acoustic Transducers.

Software: LabVIEW, MATLAB.

Keywords: Data Acquisition, Signal Conditioning, Fault Diagnostics.
