
**Passive Radar System
Khalifa University | RF & Signal Processing Research

📌 Overview
Implementation of a Passive Radar (Illuminator of Opportunity) system using ambient RF signals (FM/DVB-T) for covert target detection.

⚙️ Technical Highlights
Dual-Channel Architecture: Synchronization of Reference and Surveillance channels for bistatic sensing.

Signal Processing: * Used Cross-Correlation to calculate time-delay and Bistatic Range.

Applied Clutter Cancellation to remove Direct Path Interference (DPI).

Maximized SNR through coherent integration for low-RCS targets.

Output: Generated Range-Doppler Maps to visualize target distance and velocity.

🛠️ Tech Stack
Software: MATLAB, LabVIEW, 3D Fusion.

Hardware: SDR (Software Defined Radio), RF Antennas, 3D printer.

Key Concepts: Doppler Shift, Ambiguity Functions, SNR Optimization.**
